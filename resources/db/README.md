# Reference Databases

This pipeline uses the SILVA 16S rRNA training set.

## Required files
- silva_nr99_v138.2_toGenus_trainset.fa.gz
- silva_nr99_v138.2_toGenus_trainset.fa.gz.md5

## How to obtain
Download from: https://www.arb-silva.de/

## Versioning
Database version is recorded in:
- config/config.yaml
- per-sample call.json provenance
